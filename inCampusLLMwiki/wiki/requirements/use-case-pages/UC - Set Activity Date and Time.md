# Set Activity Date and Time

Status: Draft sourced use case; DFD modeling clarified.

Scope: MVP.

Initiating actor: Student host.

Source user story: US-25.

Related requirements: FR-2501, FR-2502, FR-0402, FR-0404; NFR-10, NFR-41.

Goal: Specify the scheduled date and start time of an activity during activity creation so other students can understand when it will happen.
